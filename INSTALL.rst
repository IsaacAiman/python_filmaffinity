Installation
============

Pip
***
::

    pip install python_filmaffinity


From Source
***********

::

    git clone git@github.com:sergiormb/python_filmaffinity.git`
    cd python_filmaffinity`
    python setup.py install`


Requirements
**********************

::

    requests >= 2.0.1`
    bs4 >= 0.0.1`
